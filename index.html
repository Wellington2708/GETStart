GETStart - BootCamp DIO – Programming Logic

Este site está estruturado de acordo com o aprendizado adquirido no Bootcamp GetStart: Lógica de Programação da DIO(Digital Innovation One).
Pode ser utilizado para revisão de matéria, aquisição de conhecimento e reforço da base de lógica de programação.
Também é um projeto aberto para contribuição dos amantes da programação e desenvolvimento de sistema, aceitando assim
colaboração dos usuários interessados.
O repositório se encontra no link Github, disponibilizando através dele, toda estruturação de pastas do projeto, um arquivo README.md totalmente detalhado,
e os códigos desenvolvidos com a linguagem Javascript.

Seguem abaixo os assuntos abordados:

Tipos de Variáveis

  • String: Armazena texto
  • Number: Armazena números
  • Boolean: Armazena valores lógicos: verdadeiro ou falso
  • Inteiros: Armazenam números não quebrados (sem casas decimais)
  • Ponto Flutuante: Armazena números com casas decimais
  • Caractere: Armazena somente uma letra (aspas simples)
  • Date: Armazena data e hora

Vetores e Matriz

  • Variável: Armazena uma informação simples
  • Vetor: Armazena várias informações do mesmo tipo
  • Matriz: Armazena vários vetores com tipos variados

Operadores e Expressões

Operadores Aritméticos

  • Adição: +
  • Subtração: -
  • Multiplicação: *
  • Divisão: /
  • Módulo: %

Operadores Relacionais

  • Igual a: ==
  • Diferente de: !=
  • Maior que: >
  • Menor que: <
  • Maior ou igual: >=
  • Menor ou igual: <=

Operadores Lógicos

  • AND: &&
  • OR: ||
  • NOT: !

Operadores de Atribuição

  • =
  • +=
  • -=
  • *=
  • /=
  • %=

Incremento e Decremento

  • Incremento: ++
  • Decremento: --

Estruturas Condicionais

São estruturas que permitem ao programa tomar decisões, executando diferentes blocos de código dependendo se uma condição é verdadeira ou falsa.

  • if: Executa um bloco de código somente quando a condição for verdadeira
  • else: Executa um bloco de código quando a condição do if for falsa
  • else if: Permite testar novas condições caso o if inicial seja falso
  • switch: Compara um valor com diversos casos possíveis e executa o bloco correspondente

Laços de Repetição

Permitem executar um código repetidamente enquanto uma condição for verdadeira ou até um número de repetições ser atingido.

  • for: Usado quando se sabe quantas vezes o bloco deve ser repetido (inicialização, condição, incremento)
  • while: Repete o bloco enquanto a condição for verdadeira
  • do...while: Executa o bloco pelo menos uma vez, checando a condição após a execução

Funções

Servem para reutilizar código e dividir a lógica em partes mais organizadas.

  • Pode receber ou não parâmetros
  • Pode retornar ou não um valor
  • Sempre executa uma ação
  • Funções com Parâmetros: Precisam de valores de entrada para executar a ação
  • Funções com Retorno: Devolvem um valor, permitindo armazená-lo ou utilizá-lo em outras partes do código

Estrutura de Dados e Objetos

JSON

Serve para agrupar dados de maneira organizada, usado para guardar e transportar informações entre sistemas (APIs REST), com formato de chave e valor.

Objetos e Classes

  • Classe: Padroniza a estrutura de dados e pode conter métodos
  • Objeto: Instância da classe com valores e métodos próprios
  • Instanciar um Objeto: Criar um novo objeto na memória baseado em uma classe
